## ✏️ How to Edit This Portfolio

You do not need to touch the complex React or CSS code to update your website! 

I have centralized all the content into **one single file**. To update your details:

1. Navigate to the `src/` folder.
2. Open the `data.js` file.
3. Edit the text inside (such as your bio, projects, or new skills).
4. Save the file! The website will instantly and automatically update with your new information.

---

## 🚀 How to Run Locally

To view and edit this project on your own machine, follow these steps in your terminal:

**1. Install all dependencies:**
```bash
npm install
```

**2. Start the development server:**
```bash
npm run dev
```

**3. View the website:**
Open your browser and navigate to `http://localhost:5173`.
