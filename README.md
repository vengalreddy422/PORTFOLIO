## 📝 How to Customize (End-User Friendly!)

I've designed this portfolio so that **anyone** can update their information in seconds. You **do not** need to be a React expert!

All website content is centralized in **one single file**.

1. Navigate to the `src/` folder.
2. Open the `data.js` (or equivalent content file) in any text editor.
3. Edit the plain text to update your:
   - Bio & Introduction
   - Projects & Experiences
   - Skills & Social Links
4. **Save the file.** The website will instantly update with your new information!

## 🚀 Getting Started

Want to run this project on your local machine or deploy it yourself? Follow these simple steps:

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### 1. Installation

Clone the repository and navigate into the project directory:
```bash
git clone https://github.com/vengalreddy422/vengal-reddy-portfolio.git
cd vengal-reddy-portfolio
```

Install the required dependencies:
```bash
npm install
```

### 2. Running Locally

Start the development server:
```bash
npm run dev
```
Open your browser and visit `http://localhost:5173` to view the live portfolio!

### 3. Building for Production

When you're ready to deploy, create an optimized production build:
```bash
npm run build
```
This will generate a `dist/` folder containing the deployable assets.
